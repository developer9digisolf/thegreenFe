
# 🚀 Next  Architecture

Proyek ini adalah boilerplate/arsitektur dasar berbasis **Next.js** untuk memulai pengembangan aplikasi web modern dengan integrasi state management, UI framework, dan utilitas tambahan.  

## 📦 Teknologi yang Digunakan

### Core
- **[Next.js 14](https://nextjs.org/)** → Framework React dengan dukungan server-side rendering, static generation, dan routing bawaan.
- **[React 18](https://react.dev/)** → Library UI untuk membangun antarmuka interaktif dan modular.
- **[TypeScript](https://www.typescriptlang.org/)** → Superset JavaScript untuk type-safety dan pengembangan lebih terstruktur.

### State Management
- **[Zustand](https://zustand-demo.pmnd.rs/)** → State management ringan, sederhana, dan performa tinggi.

### UI & Styling
- **[Ant Design](https://ant.design/)** → UI component library dengan desain modern.
- **[Tailwind CSS](https://tailwindcss.com/)** → Utility-first CSS framework untuk styling cepat dan konsisten.
- **[Sass](https://sass-lang.com/)** → Preprocessor CSS untuk styling yang lebih fleksibel.

### Utilities
- **[Axios](https://axios-http.com/)** → HTTP client untuk komunikasi API.
- **[Crypto-JS](https://github.com/brix/crypto-js)** → Library untuk enkripsi dan dekripsi data.

### Development Tools
- **ESLint** → Linter untuk menjaga kualitas kode.
- **PostCSS** → Alat transformasi CSS modern.
- **@types (TypeScript Definitions)** → Dukungan type definitions untuk React, Node.js, dan Crypto-JS.
